# 🚀 Nexalyst: Smart Consultation & Quoting System

Sistem konsultasi digital otomatis yang dirancang untuk membantu klien melakukan simulasi kebutuhan proyek website sekaligus mendapatkan estimasi harga secara real-time.

## 🌟 Fitur Utama
- **Multi-step Consultation:** Alur pertanyaan yang sistematis untuk menggali kebutuhan klien.
- **Smart Logic Engine:** Menghitung estimasi harga berdasarkan fitur, tingkat kesulitan, dan budget klien.
- **Admin Control Panel:** Panel khusus untuk mengubah harga layanan secara dinamis tanpa menyentuh kode.
- **WhatsApp Integration:** Output hasil konsultasi otomatis terkirim ke WhatsApp sebagai pesan rapi.
- **Modern UI:** Desain *Glassmorphism* dengan gradasi Navy ke Sky Blue menggunakan Tailwind CSS.

## 🛠️ Teknologi yang Digunakan
- **HTML5** & **Vanilla JavaScript** (Logic & Engine)
- **Tailwind CSS** (Styling & Layout)
- **Local Storage** (Data Persistence tanpa database luar)
- **GitHub Pages** (Hosting)

## 📂 Struktur File
- `index.html`: Halaman utama untuk klien (Landing Page & Form).
- `admin.html`: Gerbang kontrol untuk pemilik sistem (Update Harga).
- `script.js`: Otak sistem yang mengatur kalkulasi dan keamanan.

## 🔐 Cara Akses Admin
1. Buka file `admin.html` melalui browser (Contoh: `username.github.io/repo/admin.html`).
2. Masukkan password default: `admin123`.
3. Atur harga jasa Anda di panel yang tersedia dan klik **Simpan Perubahan**.
